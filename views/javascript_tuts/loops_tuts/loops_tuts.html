while loop with break: Any loop (for,while) can be terminated by break statment while(condition) will terminate if condition is false, undifinded, null or 0 value but even if the condition is set we can break it the loop by:

--------------break the loop--------------------------- var value = 3; while(value){ if(value==1) break; console.log('value',value); value--; } //same with for loop; var value = 3; for(value;value;value--){ if(value==1) break; console.log('value',value) } ------------------------------------------------------------
