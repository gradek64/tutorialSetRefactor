Fr unit 1fr, 2fr etc

An interesting unit is 1fr or even 0fr which basically means 'take free space', but you can multiply free space as 1fr, 2fr etc. 1fr 90px 1fr === auto 90px auto ( means stretch first and last) //the size of 1fr defines how much will be any multiply of 1free 2fr 10% 1fr === 65% 10% 25%; ( means stretch first and last by the width of 2 x first) 3fr 10% 1fr === 65% 10% 25%; ( means stretch first and last by the width of 3 x first) 0fr 90px 10fr === auto 90px auto ( means min first and last becomes auto cause we multiply by 0 so gives 0 so it treated as auto) this represents 3 columns first auto stretched, second 5px wide , third auto stretched grid-template-rows: [first] 20% [second] 90px [last] auto ; this represents 3 rows first 20% height, second 90px height , last auto stretched vertically
