flex:1 is combination of flex-grow, flex-shrik and flex-basis and allows you to strech content to fill up the space as the search bar below;

flex-grow will strech accoding to available space

flex-shring will shrink accoding to available content

flex-basis create a base width from where flex-grow and flex-shrink will work from

there some common settings to the flex-grow, flex-shrink, flex-basis

flex:1 0 200px = means element will grow with min width of 200px

flex:0 1 200px = means element will not grow with max width of 200px

flex:0 0 200px = means element will not grow and will not shrink setting fixed width:200px

flex:0 0 auto = means element will not grow and will not shrink and will take content width

Understanding flex-shrink

flex-shrink will adopt to content of the element not the remaining spaces as it is in flex-grow

the golden rule is if you have flex-shrink: 0 that means it wont shrink , if you have flex-shrink:1 then it will shrink , the best result can be used with flex-grow and flex-basis
