CSS centering and vertical positioning

inline-block

About:

It will only center it vertically using text-align: center; (parent) and display: inline-block; (child)

CSS

inline-centering { border: 1px solid green; text-align: center; width:100%; height:200px; }
.inline-centering .centering { border:1px dotted gray; display: inline-block; color:orange; width: 20%; }
