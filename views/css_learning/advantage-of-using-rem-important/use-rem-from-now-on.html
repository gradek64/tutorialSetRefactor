The great think about using rem every in a page is for scaling reason, 1rem == default browser font size!!
Thefore set you browser size to 10px , most browser have 16px so you need around font-size: 62.5%; and that would give U a 10px as default, next use rem everywhere for paddings,margin for fonts etc so If U ever want to increase your font all you habe to do is to change default body {font-size: 100%} so it will give you back 16px;
